デザイナーへ

visual kerning は、CSS の letter-spacing では実現できない1文字ペアごとの間隔調整をブラウザ上で行えるツールです。

エンジニアに依頼する場合

以下をコピーしてエンジニアに送ってください。

見出しやキャッチコピーのカーニングを調整したいです。
visual kerning というライブラリで、ブラウザ上でカーニングを調整してJSONに書き出せます。
書き出したJSONを本番に適用する手順はREADMEに書いてあります。

https://github.com/ie3jp/visual-kerning/blob/main/README.ja.md

自分でHTMLを書き出す場合

  1. 上部サンドボックスにテキストを入力し、フォントやサイズを設定する
  2. OptionAlt + ←/→ で文字間を調整する（±1 / ±10 / ±100 は Shift / 無し / CmdCtrl で切替）
  3. HTMLを書き出す でカーニング済みHTMLをダウンロードする

書き出したHTMLはそのままWebページに貼り付けて使えます。

エンジニアへ

外部依存ゼロ、フレームワーク非依存。ESM のみ。

npm install visual-kerning
yarn add visual-kerning
pnpm add visual-kerning
<script type="module">
  import { visualKerning } from
    'https://esm.sh/visual-kerning@0.11.0'
</script>

使い方

開発・ステージング環境で編集

import { visualKerning } from 'visual-kerning'

const editor = visualKerning({ editable: true })
editor.mount()

本番環境で適用

const editor = visualKerning({
  editable: false,
  kerning: kerningData
})
editor.mount()

開発・ステージング環境で編集

<script type="module">
import { visualKerning } from
  'https://esm.sh/visual-kerning@0.11.0'

const editor = visualKerning({ editable: true })
editor.mount()
</script>

本番環境で適用

<script type="module">
import { visualKerning } from
  'https://esm.sh/visual-kerning@0.11.0'

const kerningData = await
  fetch('/kerning-export.json')
    .then(r => r.json())

const editor = visualKerning({
  editable: false,
  kerning: kerningData
})
editor.mount()
</script>
