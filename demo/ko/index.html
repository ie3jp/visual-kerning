엔지니어에게

외부 의존성 없음, 프레임워크 독립적. ESM 전용.

npm install visual-kerning
yarn add visual-kerning
pnpm add visual-kerning
<script type="module">
  import { visualKerning } from
    'https://esm.sh/visual-kerning@0.11.0'
</script>

사용법

개발/스테이징 환경에서 편집

import { visualKerning } from 'visual-kerning'

const editor = visualKerning({ editable: true })
editor.mount()

프로덕션 환경에 적용

const editor = visualKerning({
  editable: false,
  kerning: kerningData
})
editor.mount()

개발/스테이징 환경에서 편집

<script type="module">
import { visualKerning } from
  'https://esm.sh/visual-kerning@0.11.0'

const editor = visualKerning({ editable: true })
editor.mount()
</script>

프로덕션 환경에 적용

<script type="module">
import { visualKerning } from
  'https://esm.sh/visual-kerning@0.11.0'

const kerningData = await
  fetch('/kerning-export.json')
    .then(r => r.json())

const editor = visualKerning({
  editable: false,
  kerning: kerningData
})
editor.mount()
</script>
